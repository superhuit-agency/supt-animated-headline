== Description ==
A simple animated headline shortcode. Simplyfied and based on https://wordpress.org/plugins/animated-headline/

= Example: =
[supt-animated-headline titlre="Hello" animated_text="world,you,buddy"]

== Installation ==
1. Upload and unzip `supt-animated-headline.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode in your post/page content
